# 타입스크립트-네코랜드루아스크립트 예제

## 개발환경 
git, node.js , npm 필수


### 프로젝트 복사
`git clone https://github.com/mogunogu/typescript-nekoland-example.git example_project`


### 패키지 설치
`cd`
`npm run install`

### 빌드

#### 서버
`npm run build-server`

#### 클라이언트
`npm run build-client`

#### 서버&클라
`npm run build-client`
